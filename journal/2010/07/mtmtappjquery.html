MTの管理画面を「MTAppjQueryプラグイン」で簡単にカスタマイズ

2010年07月05日 Movable Type 柳谷 真志

みなさん、こんにちは、まーしーです

今日たまたまtwitterをながめた時に（たまたまですよ）、Tinybeansさんが

MTAppjQuery 0.01 リリース - Movable Type 5 の管理画面をjQueryでカスタマイズしやすくするプラグイン (http://tinyurl.com/2awb3ld)

http://twitter.com/tinybeans/status/17757718948

こんなことをつぶやかれてたもんだからはやいところ触ってみなければ、と言うことで早速入れてみました。

MTAppjQueryプラグインをいれてみる

プラグインを入れてみるとこんな感じになっています。

  • 入力欄のラベルカスタマイズ
  • マウスオーバーしたときのメッセージ
  • その他メッセージ表示

ができるようです。 いやはや、すごいですね。

Tinybeansさんが以前作られてた

Movable Type のブログ記事編集画面をウィザート形式にするプラグイン - Edit Entry Wizard - かたつむりくんのWWW
http://www.tinybeans.net/blog/2009/07/17-172127.html  

とかもすごいなぁ、と思ってはいたんですが今回のMTAppjQueryプラグインで管理画面が色々出来ちゃいますね。管理画面のテンプレートを触るよりは格段にわかりやすいです。

MTAppjQueryに同梱されているプラグイン

MTAppjQueryはMTの管理画面をjQueryで簡単にカスタマイズすることが出来る様になります。必要なjQueryプラグインを読み込ませたりといったかゆいところにも手が届いています。

MTAppjQuery自体もいくつかjQueryのプラグインを同梱してあります。

  • $.MTAppCustomize();
  • $.MTAppMultiCheckbox();
  • $.MTAppMsg();

どれもこれも気になる所ですが、個人的には入力欄の並び替えが簡単にできるという所が一番便利そうな気がしています。

$.MTAppFieldSort({sort: 'keywords,c:textfield,title'});

サンプルファイルを見てみると、このような記述をuser.jsにすることで並び替えが自由に出来ます。本文／追記の所は「text」と書けばいいようですね。

カスタムフィールドにも対応していて至れり尽くせりです。
すばらしいプラグインですね！
