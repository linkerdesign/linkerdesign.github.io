MT5を使うなら『Movable Type 5実践テクニック』

2010年03月25日 Movable Type 柳谷 真志

みなさん、こんにちは、まーしーです。
桜が咲き始めたのに寒いですね〜。温度差が結構あるので体調管理には細心の注意を払いたいところです。

『Movable Type 5実践テクニック』を書きました

この度、Movable Type 5の本を書きました。共著者の端くれとして参加させてもらいました。
今日書店に立ち寄ったら既に売り出されてて、気づいたらなぜか手にとってレジに並んでました（笑

Movable Type 5実践テクニック
蒲生 トシヒロ, 藤本 壱, 小野崎 直昭, 難波田 実香, 柳 泰久, 柳谷 真志, 丹羽 章
ソフトバンククリエイティブ ( 2010-03-26 )
ISBN: 9784797358834
mersy's bookshelfで詳細を見る
MediaMarker

著者：蒲生トシヒロ、藤本壱、小野崎直昭、難波田実香、柳泰久、柳谷真志、丹羽章

MT5になっていくつか機能が追加されましたが、そのなかでもテーマをメインに取り扱った一冊です。
3つのサンプルサイトを例にあげてテーマの作り方を中心にMT5のカスタマイズ方法を解説しています。

MT5の本はいくつか出てきていますがテーマについてここまで突っ込んだ本はないんじゃないでしょうか？
テーマを作ったら配布するわけでそのために気をつけるポイントなども細かく書かれています。

蒲生さんが企画されその熱い思いが凝縮された一冊かと思います。それを支える共著人もそうそうたるメンツでここに加えてもらえただけでも幸せ者だと思っています。

蒲生さんの目の付け所と行動力はほんとすごいなとあらためて思わされました。

目次を眺める

目次はこんな感じです

Chapter 1 これだけは覚えよう
Chapter 2 自己紹介を作ってみよう
Chapter 3 企業案内を作ってみよう
Chapter 4 カタログサイトを作ってみよう
Chapter 5 さらにカスタマイズするために
Appendix

こんなかんじでテーマを使う場合に気をつけるべきところにまで手が届いています。

ということで

MTを使っている人は、MT5になって色々覚えることも増えたわけですがこの本を読めばテーマについては十分理解できると思います。自分自身をふくめて、MTをつかって仕事をしている人には必携の一冊かと思います。

書店で見かけた折には是非パラパラと目を通してみてください！
