こんぶだしmeeting19杯目、CSS Nite in Ginza, Vol.54のそちら側とこちら側

2010年11月22日 その他 柳谷 真志

みなさん、こんにちは、まーしーです
明日は日本では祝日なんですね。今日も休みをとって四連休という方もいらっしゃるんでしょうか？

さて、先日のCSS Nite in Ginza, Vol.54にお越しいただいた皆様、どうもありがとうございました！


［撮影：飯田昌之］

こちらからは会場の様子などを知ることができませんでしたので、どういった雰囲気だったのかわからないところが多いですが、何か考えるきっかけを提供できていれば幸いです。

当日の資料類は@cremaのエントリーで公開されていますのでそちらもご覧になっていただければと思います。

CSS Nite in Ginza, Vol.54「これがフリーの生きる道」資料公開します。｜linker journal｜linker
/journal/2010/11/css-nite-in-ginza-vol54.php

今日はこんぶだしmeeting19杯目とCSS Nite in Ginza, Vol.54でこちら側からみえていた世界をお伝えします。

収録している側とSkypeのこちら側

先日、長谷川恭久さんをお迎えしたこんぶだしmeeting19杯目でもSkypeでつなぎながら収録しました。

こんぶだしmeeting19杯目：長谷川恭久さんに、ポッドキャストのお話を伺いました。｜linker journal｜linker
/journal/2010/11/meeting19.php

収録したときの様子はこちらのエントリーでも紹介されていますが、実際に収録している側からみるとこのような感じだったようですね。

こちら側から見えていた世界はこのような感じでした。

こちらのネット環境の調子が悪く接続が切れてしまったりもしましたが、基本的にこのような感じでほぼその場にいるような感じで参加できていました。

タイムラグがあるのかどうなのか、こちらからはわからなかったのですが、聞いてみている感じだとそんなに問題なさそうでしたね。

CSS Niteの場合

CSS Niteの場合は会場ではこのような感じだったんですね。


［撮影：飯田昌之］

こちらから見えていた世界はこのような感じでした。

会場にいらした方の雰囲気というのが見えないのはなかなか難しいところでもありますが、見えない分緊張しにくいのかもしれないですね。
そう言いながらもこちらでは家で１人緊張してどうしたものか、とそわそわしてました（苦笑

映像が切れたり、音声が聞こえにくかったりしたのでこちらの声などが問題無く届いていたのか心配ではありますが、おおむね問題は無かったようですね。
面白いことに大阪との間は問題無くつながっていました。

そんなかんじで、会場とこちらから見た世界を少し並べてみました。

これからのlinkerはしばらくの間こういう感じが続いていくと思いますし、これが普通になっている人たちも多いんだろうな、と思いました。  
色々試行錯誤しつつ、改善しておかないといけないなぁ、というところもあるので色々やっていきたいところです。

CSS Niteの参考にこんぶだしmeeting19杯目もどうぞ

前述のこんぶだしmeeting19杯目の内容も、CSS Niteでの話に近いところもあるので、もしお聞きになられていない方はお時間があるときに聞いてみてはいかがでしょうか？
ヤスヒサさんのフリーランス話などは参考になる方が多いかもしれません。
