ウェブ制作にあると便利なGoogle ChromeのExtensions（拡張機能）、あれこれ。

2010年05月08日 その他 秋野 琢

こんにちは、taku-workです。
最近Firefoxの調子がわるく、メモリは食うわCPUは使うは良く落ちるわで、ほんとどうしてくれよう。

ということで、今回はウェブ制作にあると便利なGoogle ChromeのExtensions（機能拡張）のおはなし。
Extensionsが充実しているならブラウザの乗換も、とお考えの方などにどうでしょうか。

Web Developer

Web Developer - Google Chrome 拡張機能ギャラリー
https://chrome.google.com/extensions/detail/bfbameneiokkgbdmiekhjnmfkcnldhhm

Web開発者にはおなじみのWeb DeveloperのGoogle Chrome版
相変わらずの便利さです。

MeasureIt!

MeasureIt! - Google Chrome 拡張機能ギャラリー
https://chrome.google.com/extensions/detail/aonjhmdcgbgikgjapjckfkefpphjpgma

firefoxでもおなじみのブラウザ上で任意の範囲のピクセル数を計れるツール。

Explain and Send Screenshots

Explain and Send Screenshots - Google Chrome 拡張機能ギャラリー
https://chrome.google.com/extensions/detail/mdddabjhelpilpnpgondfmehhcplpiin

見ているページのキャプチャーを取ることができます。
キャプチャーを取った後に矢印や文字を入れるといった多少の画像編集ができます。

Eye Dropper

Eye Dropper - Google Chrome 拡張機能ギャラリー
https://chrome.google.com/extensions/detail/hmdcmlfkchdmnmnmheododdhjedfccka

サイトの色をピックアップできるカラーピッカー。
firefoxのアドオンでいうところのColorZillaのようなことができます。

Firebug Lite

Firebug Lite - Google Chrome 拡張機能ギャラリー
https://chrome.google.com/extensions/detail/bnbbfjbeaefgipfjpdabmpadaacmafkj

こちらもfirefoxでもおなじみのFirebugのGoogle Chrome版
Google Chromeにもともと付いているデベロッパーツールと若干かぶりますが。

Resolution Test

Resolution Test - Google Chrome 拡張機能ギャラリー
https://chrome.google.com/extensions/detail/idhfcdbheobinplaamokffboaccidbal

ウィンドウサイズを1024×768や800×600などに表示しなおしてくれます。
元ページを残したまま別ウィンドウで開いてくれるのが地味に便利。任意のサイズも指定できます。

Table Capture

Table Capture - Google Chrome 拡張機能ギャラリー
https://chrome.google.com/extensions/detail/iebpjdmgckacbodjpijphcplhebcmeop

テーブルのあるサイトではURLのバーの右にアイコンが表示されます。
アイコンをクリックするとテーブルのコピーやGoogle Docsへのはりつけができます。

Create Link

Create Link - Google Chrome 拡張機能ギャラリー
https://chrome.google.com/extensions/detail/gcmghdmnkfdbncmnmlkkglmnnhagajbm

ページのタイトルやURLのリンクを作成してくれます。
任意の組み合わせが作成できるので、タイトルとURLをはりつけが楽になります。ブログ書く時にももってこい。
firefoxでは同じ機能のアドオン「MakeLink」があります。

Chrome Sniffer

Chrome Sniffer - Google Chrome 拡張機能ギャラリー
https://chrome.google.com/extensions/detail/homgcnaoacgigpkkljjjekpignblkeae?hl=ja

ページで使用しているライブラリやフレームワークを表示してくれます。
ソースを見なくてもどのライブラリかわかるのは案外便利。

ということで

Extensions（拡張機能）もだいぶ増えているようでいい感じ。やはりきびきび動くブラウザは良いですね。
あとは1passwordがdev版だけではなく、正式に使えるようになればと思うのですが。
