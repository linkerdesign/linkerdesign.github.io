ブログを書くのによく使うアプリたち

linker journal » その他 » ブログを書くのによく使うアプリたち

こんにちは、taku-workです。
よくよく考えてみると、このlinker journalも一年以上書いているのでそれなりに慣れてきているのだなぁと思うわけで、ちょっとは身になっているのではないのかなと思ったりもしています。
とりあえず水曜と土曜の夜には「ブログ書かなきゃ」が口癖に。

ということで、今回はブログを書くときによく使うアプリのお話です。
ブログを書いている方ならおなじみのものも多いですが何かの参考になればと。

ecto

illumineX :: ecto - blog editor for Mac OS X
http://illuminex.com/mac/ecto/

ブログを書くエディターとして非常に役に立ちます。
おおよそブログを書くのに必要な機能は入っているかと。
linker journalでもまーしーさんがいくつか記事を書いていますので参考にどうぞ。

ectoのショートカット登録の話｜linker journal｜linker
http://linker.in/journal/2009/09/ecto.php

ectoのBookmarkletをつかって簡単投稿｜linker journal｜linker
http://linker.in/journal/2010/03/ectobookmarklet.php

Skitch

Skitch.com + Skitch = fast and fun screen capture and image sharing.
http://skitch.com/

mac使いにはおなじみのキャプチャーアプリです。
こちらも使い勝手が良いのでブログにキャプチャーを使う時などに良く使います。
Frameを決めて時差付きでキャプチャーできるのは便利。
あとドラッグでファイル保存できるのも便利です。

以前書いたSkitchの記事もありました。参考までに。

スクリーンショットや画像編集に便利なSkitchのコネタ｜linker journal｜linker
http://linker.in/journal/2009/03/skitch.php

ClipMenu

ClipMenu: A clipboard manager for Mac OS X - ClipMenu.com
http://www.clipmenu.com/

クリップボードを拡張してくれるアプリです。
コピーした内容も設定次第でいくつでも記憶しておけます。またスニペットとして保存しておくことも可能です。
ちょっと前にコピーしたのにという内容をフォローできるのはありがたい。

Make Link

Make Link :: Add-ons for Firefox
https://addons.mozilla.org/ja/firefox/addon/142

firefoxのアドオンです。
今開いているサイトのタイトルやURLをコンテキストメニューからコピーできます。
また、任意の設定もできるのでブログ用にHTMLタグを予め入れておくなどすると便利です。

Things

Things - task management on the Mac
http://culturedcode.com/things/

これはGTDにつかうアプリです。
直接ブログを書くことに役に立つわけではないのですが、ブログはついつい書くのを忘れてしまうのでこういったアプリで毎週のタスクとしてスケジュールに登録しておくと、うっかり忘れも少なくなります。

ということで

キャプチャーやコピペあたりはツールでずいぶん楽になりますね。
肝心の内容のほうですが、こればかりは脳みそに頑張ってもらうしかないようです。
