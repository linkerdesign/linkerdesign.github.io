web制作のとき、たまに使う便利なツール

linker journal » その他 » web制作のとき、たまに使う便利なツール

こんにちは、taku-workです。
これはまたいつか読むかもと思う本ほど読まなかったりするわけで、でもデータ化する程でもないかなぁ、いっそ処分するかなぁ。でもこのページだけ、たまに読むかなぁ。
といった一連の本の仕分け作業は、夜中に始めちゃいけません。

というわけで、今回はweb制作のとき、たまに使う便利なツールのおはなし。
いつも必要なわけじゃないけど、たまに必要になる。そんな感じのツールです。

gridr buildrrr

gridr buildrrr
http://gridr.atomeye.com/

カラム幅を計算するツールです。
地味ながらありがたいツールです。
外側のマージンとカラム間のマージンを別指定できるのが地味ながら良い感じ。

正規表現サンプル(正規表現チェッカー）

正規表現サンプル(正規表現チェッカー)
http://hodade.adam.ne.jp/seiki/page.php?checker2

正規表現を文字列に一致するかをチェックできます。
正規表現ってどうも忘れちゃうのでたまにお世話になります。

エンコードマニアックス - 各種エンコードやハッシュを一発作成

エンコードマニアックス - 各種エンコードやハッシュを一発作成
http://encodemaniax.com/

文字列を各種のコードなどに変換してくれるツールです。
実体参照など、ついつい忘れちゃうのでこちらもたまにお世話になります。

LinkChecker - Firefox link validator extension for web developers

LinkChecker - Firefox link validator extension for web developers
http://www.kevinfreitas.net/extensions/linkchecker/

ページのリンク状態をチェックするfirefoxのアドオンです。
インストールしてコンテキストメニューから「Check Page Links」をクリックすると表示されているページ内のリンクの状態を色別で表示してくれます。
リンク切れ箇所をチェックするのにたまに使えます。

ということで

いつも使うわけではないのですが、たまに使いたい時がある、そんなツール達でした。
あれ、なんだっけとなったときにほんの少しだけ手間が減ります。ほんのすこしだけ。
