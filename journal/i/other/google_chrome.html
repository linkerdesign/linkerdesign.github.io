Google Chromeに入れてる拡張あれこれ

linker journal » その他 » Google Chromeに入れてる拡張あれこれ

みなさん、こんにちは、まーしーです
今日まで入谷では朝顔市。今年はいけませんでしたがこういう祭りがあると季節を感じることができていいですね！

以前、@taku-workが「ウェブ制作にあると便利なGoogle ChromeのExtensions（拡張機能）、あれこれ。」というのを書いていました。
最近ブラウザを用途によってわけるかんじにしているので自分で入れているのはなんだろう？というのをとりあえず見直してみたいと思います。

@taku_workが既に紹介した下記の物は省きます。

  • Web Developer
  • MeasureIt!
  • Explain and Send Screenshots
  • Eye Dropper
  • Firebug Lite
  • Resolution Test
  • Create Link
  • Chrome Sniffer

では見ていきます。

1Password Extension

Setup instructions
http://support.agilewebsolutions.com/showthread.php?22003-Setup-instructions

1Passwordを使えるようになる拡張ですね。Webサイトを見る際には欠かせません。

Diigo Web Highlighter and Bookmark

Diigo Web Highlighter and Bookmark for Chrome | Diigo
http://www.diigo.com/tools/chrome_extension  

Diigoにポストするのが簡単にできるようになります。

Taberareloo

Taberareloo
https://chrome.google.com/extensions/detail/ldcnohnnlpgglecmkldelbmiokgmikno  

DiigoのプラグインはDiigoにしかpostできませんが、こちらはFirefoxのTomblooと同様で色々なSBMなどにポスト出来ます。

Evernote ウェブクリッパー

Evernote ウェブクリッパー
https://chrome.google.com/extensions/detail/pioclpoplcdbaefihamjohnefbikjilc  

Evernoteに簡単にクリップできます。Evernoteが占める位置は結構あがってきているきがするので必須ですね。

Feedly

Feedly
https://chrome.google.com/extensions/detail/ndhinffkekpekljifjkkkkkhopnjodja    

Google Readerを読みやすくしてくれるFeedlyの拡張です。

HTML Validation

HTML Validation
https://chrome.google.com/extensions/detail/epoepjgkmfbjagiemjadbjdbappjdgjo  

HTMLをValidateしてくれます。似たような拡張はいくつかあるようです。

Pendule

Pendule
https://chrome.google.com/extensions/detail/gbkffbkamcejhkcaocmkdeiiccpmjfdi

Web制作に必要そうなものが色々詰まっています。

SBMカウンタ

SBMカウンタ
https://chrome.google.com/extensions/detail/pdoiecdfkkomendcbkpddhlklnompmbn

見ているページがどのくらいブックマークされているかを知ることが出来ます。

Shortcut Manager

Shortcut Manager
https://chrome.google.com/extensions/detail/mgjjeipcdnnjhgodgjpfkffcejoljijf  

キーボード派には必須ですね。

webページショット - Webpage Screenshot

webページショット - Webpage Screenshot
https://chrome.google.com/extensions/detail/ckibcdccnfeookdmbahgiakhnjcddpki

Webページのキャプチャを撮ってくれます

ということで

まだまだChromeを使いこなしているわけではないですが、通常使うブラウザのメインにしようかな、と考えています。制作しているときのメインはSafariにしようかな、というかんじです。

新しいツールをつかうとこれまでと同じように出来ない分若干ストレスを感じたりする部分もありますが、ブラウザを分けることのメリットなどもありそうな気がしています。

拡張を入れすぎるとこれまでと同じ話になってしまうのでこれから少し考えておきたいところですね。
