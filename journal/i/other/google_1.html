Google カスタム検索でシノニムを設定して同義語の取りこぼしを減らしてみる

linker journal » その他 » Google カスタム検索でシノニムを設定して同義語の取りこぼしを減らしてみる

みなさん、こんにちは、まーしーです

今日はGoogle カスタム検索の中にあるシノニムを触ってみた話でも。

シノニムとは下記に若干記載があります。

より的確な結果を得るためのユーザー クエリの改善 - Google Custom Search API - Google Code
http://code.google.com/intl/ja/apis/customsearch/docs/queries.html#synonyms  

同義語を追加することで検索結果に反映することが出来るようです。

シノニムの作成

同義語の追加はこのようにして行えます。

サンプルで作ってみた検索エンジンを使い自分の日記を「ブーゲンビレア」で検索してみると、３件結果が表示されました。

個人的には「ブーゲンビリア」と記述することが多いので「ブーゲンビレア」と「ブーゲンビリア」を同義語に設定してみます。

こう設定した上で「ブーゲンビレア」で検索し直してみます。

ブーゲンビリアも検索対象になっているようです。
本来はもう少し違った感じで使うものだと思いますが、検索キーワードの幅を広げる事ができますね。

「ブーゲンビリア」と「ブーゲンビレア」くらいであればGoogleがいずれ解決してしまうと思いますが、専門的な場合などはこのようなかんじで独自に追加して設定しておくということもありそうですね。
