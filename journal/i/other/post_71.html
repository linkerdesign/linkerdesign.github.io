【年末年始休暇版】植物でも育ててみませんか？

linker journal » その他 » 【年末年始休暇版】植物でも育ててみませんか？

みなさん、こんにちは、まーしーです
先週末からlinkerは年末年始休暇になっています。
ということでここ数日のlinker journalは休暇モードで特にテーマも決めずにお届けします。

みなさん、自宅で植物とか育てたりしていますか？
昔は育てていなかったのですが、カメラをもつ楽しさを知って以来、実家の庭で育てられていた植物に興味を持つようになりました。
その影響だとおもいますが、いくつか植物を自宅で育てたりしています。

みなさんもそうかもしれませんが、ひたすらディスプレイに向かう生活なので緑が癒しをあたえてくれるかも？という期待も少なからずあるのですが今日はそんな我が家の植物の紹介でもご紹介します。

サンゴアブラギリ


沖縄のお土産でもらいました。

現在、かなり弱りモードです。葉が２枚ほど枯れ落ちるのも時間の問題ですね。
水をやるのをわすれると枯れるのが早いかもしれません。花が咲くらしいのですが、育て始めて数年間、花が咲くのを見たことがないです。

plantsサンゴアブラギリ - a set on Flickr

ハイビスカス


真っ赤な花が印象的です。
ちょうどまたつぼみができはじめてきました。
近所の園芸屋にあったのをたまたま買った感じなのですが、結構強くて、花もそれなりに咲くのでみてて飽きないですね。
育てるのにはオススメです。

plantsハイビスカス - a set on Flickr

セントポーリア


あんまり日に当てない花らしいです。
つぼみができるまでかなり時間がかかってしまって寂しいときは数年花がないかんじでしたが、つぼみができはじめるとひたすら咲いてます。

plantsセントポーリア - a set on Flickr

ベゴニア

撮った人の腕がわるいのであまりキレイに撮っていませんが花はほとんどずっと咲いてます。
そういう意味では目を楽しませてくれますね。

plantsベゴニア - a set on Flickr

トラノオ

観葉植物として目にすることが多いんじゃないでしょうか？葉っぱだけなので寂しいですが成長が目に見えてわかります。

もともと一本だったのがどんどん地下茎が増えて今は３本になってます

plantsトラノオ - a set on Flickr

ガジュマル

  

これも沖縄土産ですね。

いまのところ花も咲かず、枯れもせず、地味なヤツです。

金のなる木

よく5円玉をとおしたりすることからこう呼ばれるそうです。

ぼちぼち鉢を大きくしてやって植え替えの時期の気がします。

plants金のなる木 - a set on Flickr

ブーゲンビリア


この写真ではかなりわかりにくいのですが、この枝がのびまくって大変なことになってます。

花は結構キレイです。

plantsブーゲンビリア - a set on Flickr

ポトス


どんどん伸びていきます。
ちょっと諸事情があって一旦切ってしまったのですが、いままた再生中です。
這わせるように使ってるコードはあまってた電話のケーブルです（笑

plantsポトス - a set on Flickr

ボケ


我が家の新人です。本日からjoinしました。
なんかつぼみが沢山な気がしたので買ってきました（450円）。
花が咲くのが楽しみです。ボケという名前もこれまた、といったかんじです。

植物を育てるのは楽しいです

なにげなく始めたこれらの植物を育てたりすることなのですが、結構楽しいです。
水やりとか季節によってちがったりしますが、大幅に忘れたりしなければ枯れたりしないですね。植物は強いです。
（ベランダ菜園にも初めてチャレンジしましたが失敗しました）

上記であげた植物の中だと、ベゴニア、トラノオ、ハイビスカスあたりはそこそこ強いんじゃないかと思います。
お店とかでみて気に入ったモノがあれば思い入れもあるので良いかもしれませんし、特になければこの辺を育ててみても良いのかもしれませんね。
小学校の夏休みじゃありませんが、朝顔とかも成長が見えるので来年の夏にはオススメです。
夏の風物詩、入谷の朝顔市もオススメです。

在宅で作業をしていて外に出ない日とかがあったりする方は植物を育ててみてはいかがでしょうか？
簡単に気分転換ができると思いますよ。
