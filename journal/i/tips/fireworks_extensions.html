ということで、今回はFireworksのextensions（拡張機能）のおはなし。
ちょっと便利になるextensionをご紹介です。

Sunburst

Aaron Beall - Fireworks Guru
http://fireworks.abeall.com/extensions/autoshapes/

文字通りsunburstをつくる拡張です。オートシェイプなので調整が楽です。
新発売とかニューリリースとか謹賀新年みたいなのによくつかわれていますね。

Fireworksでくずれた角丸を元通りにするコマンド

Fireworksでくずれた角丸を元通りにするコマンド ? きんくまデザイン
http://www.kuma-de.com/blog/2009-01-20/283

長方形を角丸にしてサイズ調整で拡大とかしたときに伸びちゃった角丸を直してくれます。
ちなみに、長方形などのオブジェクトはそれ単体の時にグループ解除を実行するとアウトライン情報は破棄されます。
すると角丸をあとからつけられなくなります。その状態ではこのコマンドは効きませんのでご注意を。

Set Layer and Object Name

Adobe - Fireworks Automation
http://www.adobe.com/cfusion/exchange/index.cfm?event=extensionDetail&loc=en_us&extid=692025#

選択しているレイヤーやオブジェクトの名前を変更できます。
複数選択も可能なので同じアイテムの名前にしたいときなど良いですね。
ずいぶん昔のスクリプトのようですが、動きました。
