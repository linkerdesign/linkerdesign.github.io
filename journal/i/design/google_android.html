こんにちは、taku-workです。
先日試しに蒔いてみた芝の種ですが、蒔く量が少なかったようでうっすらとしています。
芝のシートくらいにするには結構蒔かねばならないようで。

さて、今回はGoogle Android携帯のアプリのデザインや開発を始める時に知っていると便利な情報あれこれです。ご参考にどうぞ。

Android仕様は下記URLで見ることができます。
デザイナーとしてはGUIの部分は見ておくと良いかと思います。

Android Developer
Androidに関する仕様がまとめられています。

GUIのガイドラインなどはいろいろと参考になりそうです。
それからUIのテンプレートを公開しているサイトもいくつかありますね。
UIデザインの参考に見てみてもよいかと。

User Interface Guidelines
アプリケーションアイコンのガイドラインやステータスアイコンなど記載されています。
Android GUI PSD
UIのセットがPSDファイルでダウンロードできます。
Android UI Template Kit
こちらも同じくPSDファイルでダウンロードできます。

AndroidのフォントはDroid Fontというオリジナルフォントがインストールされています。
sans、serif、monospace、などがあって、ライセンスはApache License 2.0です。
日本語もDroid Sans Japaneseというのがあります。
Android SDK内からフォントを確認してみるのも良いですね。

Droid Font
Droid Fontを紹介しているサイトです。

開発はAndroid SDKをインストールして行います。
そこまでコードを書かなくても良いプランナーやデザイナーはGUIテストツールなどを試してみるのも良さそうです。

Download Android 1.6 SDK, Release 1
macでもwinでも開発できるのはよいですね。
DroidDraw Beta
GUIテストツール。スタンドアローン版もあるのでインストールして使えます。
