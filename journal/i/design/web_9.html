webサイトを要素やパーツごとにみることができるまとめサイトなど

linker journal » デザイン » webサイトを要素やパーツごとにみることができるまとめサイトなど

こんにちは、taku-workです。
「要約して分類する」を心がけるといろいろなことがわかりやすくなりますね。

ということで、今回はwebサイトを要素やパーツごとにみることができるまとめサイトのおはなし。
webサイトのまとめサイトは多くありますが、要素ごとやパーツごとにカテゴライズされているまとめサイトです。

Elements of Design: A Web Design Showcase

Elements of Design: A Web Design Showcase
http://www.smileycat.com/design_elements/

ページ左のメニューから各要素別にアーカイブをみることができます。
数もなかなか豊富なので見応えありますね。

Ember

View all Categories :: Ember
http://emberapp.com/explore/categories/

画像共有サービス「Ember」のカテゴリーページです。
各画像にはタグもついているので検索でも探すこともできます。検索の方が細かい要素が探せて良い感じ。

Pattern Tap

Pattern Tap : Organized Web Design Collection of User Interfaces for Inspiration and Ideas. The CSS Gallery Alternative - Pattern Tap
http://patterntap.com/

毎度おなじみの「Pattern Tap」
サンプル数が多いので見応え有り。とくにtypographyやnavigationあたりの数が多くて良いですね。

UI-Patterns.com

UI-Patterns.com
http://ui-patterns.com/

こちらも有名ですね。ヘッダーメニューの「Design Patterns」からカテゴリ別に探すことができます。
細かい分類がされているのも特徴的です。

Webデザインの"ブブン"を集めたブブンデザインアーカイブ　公式サイト

Webデザインの"ブブン"を集めたブブンデザインアーカイブ　公式サイト
http://bubundesignarchive.jp/

以前「携帯サイト制作の時に勉強になるサイトとか、あれこれ。｜linker journal｜linker」で紹介した「モバイルサイトデザイン集のモバイルデザインアーカイブ　公式サイト」の運営会社さんから新しいサービスを始めたとのご連絡をいただきました。参考になる良いサイトなのでご紹介です。

日本のサイトでの事例が見れるのはありがたいですね。

見出しデザイン.com

見出しデザイン.com
http://midashi-design.com/

こちらは見出しにフォーカスしたまとめサイト。
見出しのデザイン別にカテゴライズされているので見やすいです。
同じ運営の方で「フッターデザイン.com」というフッターにフォーカスしたサイトもあります。

ということで、

自分でもサイトのパーツをLittle Snapperでまとめたりをしますが、これがなかなか手間な作業です。
なので、こういったまとめサイトは本当にありがたいですね。多いに感謝です。
