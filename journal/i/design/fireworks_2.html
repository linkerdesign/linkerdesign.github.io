Fireworksよく使うショートカットキー、あれこれ。

linker journal » デザイン » Fireworksよく使うショートカットキー、あれこれ。

こんにちは、taku-workです。
今回はFireworksのショートカットキーのおはなし。
覚えておくとほのかに作業スピードが上がるかも。

変形の繰り返し「command + Y」

直前の変形等の操作を繰り返しておこないます。
移動はもとより、拡大縮小、色の変更も繰り返せます。

属性をペースト「shift + option + command + V」

オブジェクトの属性を他のオブジェクトにペーストできます。
塗りや線の情報、フィルタなどスタイルごとまるっとペーストできちゃいます。
これホント便利。

整列系「option ＋ command ＋1 〜 9」

整列パネルにもありますが、ショートカットでサクッと揃えられます。

縦方向

左揃え：option ＋ command ＋ 1
中央揃え：option ＋ command ＋ 2
右揃え：option ＋ command ＋ 3

横方向

上揃え：option ＋ command ＋ 4
中央揃え：option ＋ command ＋ 5
下揃え：option ＋ command ＋ 6

均等配置

横等幅揃え：option ＋ command ＋ 7
縦等幅揃え：option ＋ command ＋ 9

オブジェクトの重ね順の入れ替え「command + ↑ or ↓」

このあたりも便利です。
最前面へ：shift + command + ↑
最背面へ：shift + command + ↓

テキストの固定の幅を切り替え

テキストのバウンディングボックスをテキスト幅に合わせたいというときに使います。

この操作は[メニュー] → [コマンド] → [テキスト] → [固定の幅を切り替え] にあります。
デフォルトではショートカットキーは設定されていないので [メニュー] → [Fireworks] → [キーボードショートカット] でオリジナルのショートカットキーを設定します。

ということで、

変形の繰り返しとか属性のペーストあたりは結構つかいますね。
メニューやパネルからでもいいですが、ショートカットキーで楽したいところです。
