linker journal » Movable Type » 8月4日にMTDDC 2012が開催されるようですね

みなさん、こんにちは、まーしーです。
週末、名古屋でカレーうどん＋海老フライをたべてきました。うどんが太くて驚きでした。
うどん好きとしてはうどん県にも行きたいところです。

さて、8月4日にMTDDC 2012が開催されるようですね。

8月4日（土）MTDDC 2012（Movable Type Developers & Designers Conference 2012）を開催します | MovableType.jp
http://www.movabletype.jp/blog/mtddc2012.html  

開催概要は以下の様になっていますね。

日時 2012年8月4日 (土) 13:00 ? 18:00 (予定)
会場 日本マイクロソフト株式会社 品川本社 (東京都港区港南 2-16-3 品川グランドセントラルタワー)
定員 100名 (先着)
参加費 無料 (懇親会は4000円。Lightning Talks発表者は懇親会無料)
協賛 メディアスポンサーgihyo.jp
お申し込み https://www.sixapart.jp/inquiry/movabletype/mtddc2012.html

LTで話す人も募集してるようですね。

すごく行きたいのですが、8月4日は結婚式が入ってしまっていて行けず・・・

カンファレンスの内容は

  1. 開会 ごあいさつ: 関信浩 (シックス・アパート株式会社 代表取締役)
  2. Movable Typeのサステナビリティ: マネジャー 長内 毅志 (シックス・アパート株式会社 Movable Type 製品企画 マネジャー)
  3. クラウドを活用したWeb構築事業の支援プラットフォーム構想とプロトタイプのご紹介: 作村裕史 (シックス・アパート株式会社 クラウド事業開発 シニアコンサルタント)
  4. Movable Type 5.2 Overview (エンジニア目線): 高山裕司 (シックス・アパート株式会社 Movable Type 開発エンジニア)
  5. Azure と MT のフシギな関係 : 柳下剛利 (シックス・アパート株式会社 シニアコンサルタント)
  6. The New Rich Text Editor: 天野卓 (株式会社ToI企画 取締役 Movable Type 開発エンジニア)
  7. Movable Typeとスマートフォン対応技術の最新事情: 宮永邦彦 (アイデアマンズ株式会社 代表取締役 宮永 邦彦)
  8. Lightning Talks
  9. 18:30 - 懇親会 (希望者のみ)

といったところのようです。
行けないなりに気になるのはAzureの話とNew Rich Text Editorの話ですね。
