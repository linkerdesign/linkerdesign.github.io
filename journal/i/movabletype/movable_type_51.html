Movable Type 5.1リリース

さて、昨日MT5.1がリリースされていました。

大きなところはカテゴリ・フォルダの並び替えなどリスティングフレームワークでしょう。

あわせて4.29,5.05もリリースされているようです。似たようなかんじでバージョン違いがいくつか平行で出始めているのが若干分かりにくいところでもありますが、バージョンにあわせてアップデートしておかないといけませんね。

記事にカテゴリを指定するときは親カテゴリも指定する？

大した話じゃないのですが、リンク先でのプレゼン資料をみていて、ブログ記事にカテゴリを親カテゴリと子カテゴリ両方指定していました。

Movable Type 5.1 : What's New
View more presentations from Six Apart KK

ドリンクの子カテゴリであるコーヒーを指定するときに、ドリンクのカテゴリも指定していますね。

+親A
++子A-a
++子A-b
+親B
++子B-a
++子B-b
++子B-c

みたいなかんじでカテゴリがあって、子B-bを指定したいとした場合、どういう風にカテゴリを選択するのが多いんでしょうね。

個人的には子B-bのみを選択するイメージでした。
このような感じで親と子が両方ある場合は親はどちらかというと分類用に近いのかと思っていました。

この設定自体で大きく変わることはありませんが、たまたま見てそのように選択するかなぁ、とも思ったりしました。
