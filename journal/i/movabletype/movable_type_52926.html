Movable Type 5.2は9月26日！！

linker journal » Movable Type » Movable Type 5.2は9月26日！！

みなさん、こんにちは、まーしーです。
昨晩というか今朝、新しいiPhoneの発表とかあったみたいですが、何か気になるものはありましたか？？
iPadを新しいの買おう買おうとおもって買えてないので、一応気にはしてましたが特に動きはなかったみたいで。どうしようかな。

さて、現在ベータ版の提供がおこなわれている、MTの5.2が9月26日に出荷開始されるようです。

最新版「Movable Type 5.2」を9月26日より出荷開始 | Movable Type
http://www.sixapart.jp/movabletype/news/2012/09/12-1000.html

ちょうど先日、Movable Type 5.2 RC1版も公開になりましたね。

Movable Type 5.2 RC1を公開 | MovableType.jp
http://www.movabletype.jp/blog/movable_type_52_rc1.html

9/26に正式版が公開されるので、オンラインでのベータテストも9/26までのようです。
あと2週間ですね。

RC版での変更内容についてはyujiroさんが早速まとめられています。

Movable Type 5.2 RC1リリースと環境変数の追加・変更について: 小粋空間

いろいろなプラグインも5.2対応にむけてアップデートされたものが出始めてきていますね。

ベータ環境使えるのもあと2週間。
いろいろ試しておきたいところです。
