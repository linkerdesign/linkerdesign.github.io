「Movable Type 5 プロフェッショナルガイド」を献本いただきました。

linker journal » Movable Type » 「Movable Type 5 プロフェッショナルガイド」を献本いただきました。

火曜／金曜担当の@cremaです。

今日は、MT好きなら知らぬものはいない（！）「小粋空間」の荒木勇次郎さんからご著書を献本いただきましたので、ご紹介いたします（ありがとうございます！）。

その名も「Movable Type 5 プロフェッショナルガイド」。「プロフェッショナル」の言葉が入っている通り、これは相当なMovable Type上級者向けの書籍です。なんと全部で23章！

著者の荒木さんは、この書籍の紹介記事の中で、こんなふうに書かれています。

そういうわけで、順を追ってサイトをひとつ作ります、という本ではありません。この本を通じて、ピンポイントで困っていることに対して解決策がみつかるか、直接的な答えがなくても、何らかの手がかりがみつかる、そういう本になってくれれば幸甚の喜びです。

そうなのです。この本は、「Movable Type 4.2 パーフェクトガイド」の後継版と言うことで、開発時に困った時の辞書として大変活躍するかと思います。

それでは、全部の見出しを拾うと膨大になってしまうので、私が気になった見出しを幾つかピックアップしてみますね。

いかがでしょうか？　これだけでも、Movable Type開発時に悩んでいたあんなことやこんなことが解決しそうでわくわくした気分になってきませんか？　Movable Type5を使ったWebサイト制作に関わる方であれば、ぜひデスクの傍らに常備してきたい一冊だと思いました。

それにしても、先日@mersyが執筆参加＋ご紹介した「Movable Type 5実践テクニック」も、以前私が記事にさせていただいた「基本からしっかり分かるMovable Type5」もありますし、初級から上級まで、最近ますますMT5本が豊かになってまいりましたね。ありがたいことです。
