みなさん、こんにちは、まーしーです
昨日のウィンブルドン、伊達さんの試合をみてはらはらドキドキ。すごくいい試合をみせてもらいました。一度リタイアしたとはいえ、今も第一線で戦っている人はやはりスゴイですね。

さて、今週末の6月25日（土）にMTDDC Meetup TOKYO 2011が開催されます。

MTDDC Meetup TOKYO 2011（Movable Type Developers & Designers Conference）
http://book.mycom.co.jp/event/mt0625.shtml#moshikomi

参加者にはMT5.1対応の書籍『Movable Type 5.1　プロの現場の仕事術』もついてくるとのことなので、だいぶお得なイベントだと思いますね。

Movable Type 5.1　プロの現場の仕事術
荒木 勇次郎, 蒲生 トシヒロ, 藤本 壱, 西畑 一馬, 柳 泰久, 伊藤 のりゆき, 金子 順, 高橋 真弓, 天野 卓, 奥脇 知宏
毎日コミュニケーションズ ( 2011-06-30 )
ISBN: 9784839937317
mersy's bookshelfで詳細を見る

この書籍の執筆者の方々によるセミナーも非常に気になるところです。

  • Movable Type 5.1の新機能　（金子 順／シックス・アパート）
  • よくわかる条件分岐　（伊藤 のりゆき）
  • Movable TypeとFacebookの素敵な関係　（蒲生トシヒロ）
  • Movable Type＋CSS3＋jQueryで作るiPhoneサイト　（西畑 一馬）
  • EPUBプラグインで、Movable Typeで電子書籍　（藤本 壱）
  • クライアントの満足度を高める！管理画面の簡単カスタマイズ！　（奥脇 知宏）
  • もっとディープにMovable Typeを使いこなす　（荒木 勇次郎）

どんな話がきけるのかが楽しみですね。
参加できないのが非常に残念です。

このセミナーの締切が本日13:00までだそうです。
週末ですが、MT5.1に興味があって時間がある方は参加されてみてはいかがでしょうか？？
