火曜／金曜担当の@cremaです。

今日は、遊び友達の@gachi_folkさん（出版社エクスナレッジ勤務）より「Webデザイナー/コーダーのための HTML5コーディング入門」を献本いただきましたので、ご紹介いたします。

しかも、なんと4冊頂戴しました（ありがとうございます！）ので、3冊は読者プレゼントにさせていただきます。応募方法は、この記事の最後に書いておきますね。

この本は、小山田晃浩さん（@yomotsu）・外村和仁さん（@hokaccha）・向井領治さん（@mukairyoji）・伊藤学さん（@gaku）・藤本壱さん（@hjfuji）さんの共著で、これからHTML5を勉強しようとしている人向けの、基礎からの内容です。

私の個人的な印象なのですが、仕事の隙を縫ってHTML5の勉強をしようとしても、メーリングリストやWeb上で得られる情報は、割とプログラマさん向けのものが多いような気がしています。JavaScriptの知識があることは前提で、Webアプリ作成に取り組むというような。それだと、ノンプログラマな文系（？）Webデザイナーにはちょっと敷居が高く、なにかと断片的な情報しか得られていない焦りがあったのですが、そんな人にはこの本はかなりうってつけ。

まずCapter 1では、「HTML5の基本」と題して、次のような内容が語られています。

◇Chapter 1　HTML5の基本
1.1 HTML5とは何か
1.2 HTML5のマークアップ
1.3 CSS3
1.4 JavaScript API

HTMLの歴史や従来のHTMLとの違いという基礎から見ていくことができ、気になるCSS3とJavaScript APIにも触れられるので、まずHTML5の世界の概要を知るにはちょうど良い構成です。

さらにChapter 2では、「HTML5で新たに登場する考え方と要素型」というタイトルで、DOCTYPE宣言から章や見出しのマークアップ、そして外部データ埋め込みや入力フォーム関係の新要素まで、ひとつひとつが解説されています。いままでHTMLを書いた経験がある人であれば、その経験とすりあわせながら新要素を理解しやすい内容だと感じます。

◇Chapter 2　HTML5で新たに登場する考え方と要素型
2.1 文書の雛形を用意する
2.2 構造をマークアップする
2.3 語句をマークアップする
2.4 外部データ埋め込み系要素
2.5 入力フォーム

Chapter 3は、もう少し実践的です。「HTML5を利用したWebページ作成チュートリアル」ということで、実際に1ページマークアップする手順に沿って、考え方とソースが示されていきます。先輩マークアップエンジニアの頭の中を覗いているような内容なので、自分の仕事にすぐ応用できそうですよ。

◇Chapter 3　HTML5を利用したWebページ作成チュートリアル
3.1 全体の構成を検討する
3.2 文書情報を宣言する
3.3 ページのヘッダーを構築する
3.4 メインカラムを構築する
3.5 サイドバーを構築する
3.6 ページのフッターを構築する
3.7 チュートリアルのまとめ

巻末のリファレンスは、HTML5の要素一覧、属性一覧、カテゴリ一覧、CSS3のプロパティ一覧と、HTML5を使用している主なWebサイトが紹介されており、資料性も高いですね。

初心者が一冊みっちり読み込めば一通りの知識が得られ、その後デスクの脇において資料として見返すこともできる、とてもベーシックで役立つ内容の書籍だと感じました。私自身、知識がかなり断片的なので、隅々までこの本を読んでおきたいと思います。
