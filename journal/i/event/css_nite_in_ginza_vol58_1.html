linker journal » イベント » CSS Nite in Ginza, Vol.58無事終了。ありがとうございました。 #cssnite_ginza58

火曜／金曜担当の@cremaです。

昨日は、アップルストア銀座にて行われた「CSS Nite in Ginza, Vol.58」に、@taku-workと一緒に出演してまいりましたので、お礼＋軽くレポートを書かせていただきます。

セッションのタイトルは、「『ウェブデザインコーディネートカタログ』を 作って見えてきた、2011年のデザイントレンド考察」と、予告と少し変更しましたが、内容は予定通り。

先日発売の書籍『ウェブデザインコーディネートカタログ』に掲載したサンプルをさらにシャッフル。2010年から2011年にかけて日本のWebサイトを見て回った結果を、自分たちなりに5つのポイントに分け、お話しさせていただきました。

photo by Masayuki Iida


スライドと音声は下記のページで配布しておりますので、もし興味のある方はチェックしてみてくださいね。

  • CSS Nite in Ginza, Vol.58フォローアップ #cssnite_ginza58 | CSS Nite公式サイト

以下、私個人の感想を述べさせていただきます。

入場制限になるほどの大盛況

夏休みシーズンと重なったせいなのか、なんと入場制限になるほどの大盛況ぶり。まさかこの回にそんなに大勢の人にいらしていただけるとは想像していなかったのですが、さすがCSS Niteと原さんのブランド力なのだと感じ入りました。心から感謝いたします。

そして、入場できなかった皆さまには、本当に申し訳なく思っております。またの機会に、気軽にディスカッションできるチャンスなどがあれば！　なにかのイベント等で見かけられましたら、お気軽にお声がけくださいませ。

小山さん大活躍

前々からワタクシがその能弁さに着目していたキゴウラボの小山さんのお話がとても好評で、なんとその場で鷹野さんから年末のShift5への出演オファーが。小山さんに一緒にお仕事していただきたくてお願いした私としては、とても嬉しい出来事でした。Shift5でのデザイントレンド解説には、さらに小山さんがパワーアップして熱弁を振るってくださることと思います:)

黒野は司会に徹し過ぎたかも？

今回、進行上の都合および、原さんと小山さんがあまりにもお話上手ということで、黒野個人は司会進行に徹したのですが、観客の皆さまから見ると「あの人なにする人だったんだろう？」という印象もちらほら見受けられたようです。

monoe's blogさんの記事「CSS Nite in Ginza, Vol.58 に参加してきました - monoe's blog - Site Home - MSDN Blogs」では、私がいなかったことになっていて（笑）、あぁ印象薄かったのだなぁ、と反省することしきりです。

そもそも、今回の書籍執筆においても一番影の薄い首謀者で、いてもいなくてもいいぐらいの存在だったため、ちょっとこれではいかんなぁと、猛省しております。やっぱり「愛嬌だけで生きている中途半端なゼネラリスト」になってしまっているのですよね。ずるずると。

ということで、黒野個人は当分の間充電期間に入り、いままでお引き受けしている講師業を除いて、大勢の皆さまの前でお話しすることは控える日々に入ります。いつか自信を持って皆さまの前に戻ることができるようにガリ勉する予定ですので、各種勉強会やセミナー等でお見かけになりましたら生暖かく見守っていただければ幸いです。

最後に、鷹野さんがまとめてくださったツイートを貼付けておきますね。沢山の方にご来場いただき、ご意見をいただくことができて、本当に感謝しております。今後ともよろしくお願いいたします！
