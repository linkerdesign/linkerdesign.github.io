ブックマークも注釈も同期できるiAnnotateを使ってiPadで読書することにしました

2012年03月19日 Tips 柳谷 真志

みなさん、こんにちは、まーしーです。
この週末、所用とあわせて広島に旅行してきました。思ったより遠くてびっくりです。九州はもっと遠いんですよね・・・

さて、iPadで本（PDF）を読むにあたりどのアプリで読もうか？という話になります。
以前にもこの辺の話については検討したことがあります。

今回は『Webデザイナーのための jQuery入門』をiPadで読んでみます。

Webデザイナーのための jQuery入門　魅力的なユーザーインターフェースを手軽に作る (Books for Web Creative)
高津戸 壮 / 技術評論社 ( 2011-12-07 )
mersy's bookshelfで詳細を見る

iBooksでみるとこのようなかんじです。

今回はGoodReaderと iAnnotateのどちらをつかうか色々ためしてみて、現時点では iAnnotateで読んでみることにしました。

iTunes App Store で見つかる iPad 対応 iAnnotate PDF - 注釈
http://itunes.apple.com/jp/app/iannotate-pdf-zhu-shi/id363998953?mt=8

判断理由としてはブックマーク、注釈をDropboxを経由してPCでも共有出来る点です。
自分の方法がまずかったのかもしれませんが、GoodReaderでつけたブックマークはPCでPDFを開いた際には上手く見ることが出来ませんでした。

Annotation

iPadのiAnnotateで線を引いたりといった注釈はこのような感じでいれられますね。

つけたものは一覧を見ることも出来ます。

ブックマーク

次はブックマークです。ブックマークをつけるとこのような感じです。

ブックマークも一覧できます。

PCと同期する

今回のデータはDropbox経由で同期しました。
読むためにDropboxからPDFを取り込んであります。

保存してある物をサーバーにUploadしておきます。

PCで開いてみたAnnotation

サーバー（Dropbox）と同期したものをPCで見てみます。

AnnotationがPCで見れています。

リストも出すことが出来ます。

PCで開いてみたブックマーク

ブックマークも一覧して見ることができます。

このブックマーク部分の同期がうまくいかなかったので現時点ではiAnnotateを使うことにしてみようと思います。

ということで

注釈やブックマークをつけるということはメモ代わり程度ですし、読書の中の本質ではないと思います。
その本を読んでそれをどう生かしたというのが重要です。

とはいえそれらをしていく上でのサポートの1つとして、こういった機能があると思います。
これらも含めてうまいこと同期出来ない状況なのは少々残念な気もします。

epubならもう少しだけ上手くできたりもするので、早いところそういった形で読書できる様になるといいなぁと思います。

本を読むにしても一度PCからiPadへ送るといった手間が必要なのもまだまだなところですよね。
Kindle方式みたいに簡単にできる方法がはやいところ動いて欲しいなとおもいますね。
