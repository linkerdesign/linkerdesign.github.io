クリスマスにむけてAdvent Calendarがたくさんありますね

2012年11月22日 その他 柳谷 真志

みなさん、こんにちは、まーしーです。
黒い背景のエディタにしてると、ブラウザがすごくまぶしいということに気づきました。。。

さて、クリスマスにむけてAdvent Calendarが色々ありますね。

Advent CalendarについてはAdventar の説明を引用させてもらうと

Advent Calendarは本来、12月1日から24日までクリスマスを待つまでに1日に1つ、穴が空けられるようになっているカレンダーです。技術系のAdvent Calendarはその風習に習い、12月1日から25日まで1日づつ技術系のネタをブログに投稿していくというイベントです。

Adventar
http://www.adventar.org/

というかんじです。

MTのもあったので何を書くかは全然決めてませんがとりあえず登録だけしてみました。

Movable Type Advent Calendar 2012
http://www.adventar.org/calendars/26

その他にも気になる物としては「CSS Preprocessor」「Sublime Text 2」「Fireworks」あたりでしょうか。

CSS Preprocessor Advent Calendar 2012
http://www.adventar.org/calendars/1

Sublime Text 2 Advent Calendar 2012
http://www.adventar.org/calendars/20

Fireworks Lover Advent Calendar 2012
http://www.adventar.org/calendars/14

去年の技術系の Advent Calendar は技術評論社さんの電子書籍としてもまとめられていますね。

Advent Calendar 2011
https://gihyo.jp/dp/sp/advent2011

変化が早いので内容的にはふるいものもあるのかもしれませんが、目を通して見てもいいかもしれませんね。

気になる物で枠がまだある物には参加してみてはいかがでしょう。
