MacBook Airにいれたソフトあれこれ

2012年07月30日 レビュー 柳谷 真志

みなさん、こんにちは、まーしーです。
オリンピックが始まってますね〜。なんかほとんど見ないうちに終わってしまいそうな気がします。

さて、先日新しくMacBook Airに買い換えたのでそのときにインストールしたアプリのメモでも。

ユーティリティ

  • App cleaner
    アプリ削除。インストールしたアプリはいつか削除するかもしれないので。
  • The unarchiver
    解凍アプリはこちらに変更。
  • Alfred
    ランチャー。これまではAlfredとQuickSilverを使い分けていたのですが、1つで行こうかと思います。
  • Backblaze
    バックアップ。今回はTimeMachineではなくこのサービスでバックアップをしてみようかな、と。
  • 1Password
    パスワード管理。これがないともうやっていけないです・・・
  • Dropbox /SugarSync
    データ共有。少しこの2つを使い分けしてみようかと画策中
  • Skype
    仕事はSkype以外のツールはにしたいところでもありますが必要ですね。
  • Mailplane
    Gmail用。ほかのも試してみたいところですがひとまずこれまで通り。
  • ATOK
    Passportというのにしてみました。
  • Office for Mac
    ほかの代替ソフトでもいいかもしれませんが、一応これで。
  • OmniFocus
    とりあえずタスク管理系は変えずに
  • Evernote
    メモとしてはやはりつかいますので・・・
  • TextExpander
    Dropboxでsyncしてあればすぐ便利につかえますね
  • Marsedit
    ブログエディタは今回ectoからMarsEditにのりかえました。

ブラウザ

  • Chrome
  • Firefox
まぁ、追加で入れるブラウザといったらひとまずこの2つですかね。
今回はFirefoxをメインにしてみようかと思っています。

制作系

  • Mamp Pro
    さくっと動かしたいとき用ということで。
  • Codekit
    今回Lionにして使ってみたいソフトの1つ。
  • Tower / git
    データ、バージョン管理には欠かせないので。
  • Versions / subversion
    Gitでやりとり、管理することがおおいですが、Subversionも時々ありますので。
  • Transmit
    FTP。
  • Sublime Text 2
    エディタ。まだいれたまんまですが、いろいろ使いこなしたいですね。。
  • Coteditor
    エディタ。一応入れておきます。
  • Jedit
    エディタ。なんか最近は単にdiff用になってるきもしますが。
  • Adobe Creative Cloud
    月額制になりましたね。
  • Patterns
    正規表現をたしかめたり。
  • imageOptim
    画像のファイルサイズ圧縮系。

あと、今回触ってみたいと思ってるのはDayliteですね。
これでどういう風に管理できるのかを試してみようと思っています。

入れたソフトはこれまでとそんなに変わらないですね。
Lionにしていなかったので入れられなかったソフトとかがいくつかあるのでその辺はこれから使っていきたいところです。
